## Настройка соединения tls.ofukem.ru

Для настройки соединения выполните следующие шаги:

1. Откройте континент TLS-клиент.
2. Нажмите "Добавить".
3. Укажите следующие параметры:

| Добавление ресурса |
| :----------------- | :-------------- |
| **Адрес:**         | `tls.ofukem.ru` |
| **Имя ресурса:**   | `tls.ofukem.ru` |
| **Порт ресурса:**  | `6005`          |
| **Тип:**           | `Туннель`       |
